
# House Price Prediction using Linear Regression

## Overview
This project is part of my Machine Learning Internship at SkillCraft Technology.

The goal of this project is to predict house prices using Linear Regression based on:
- Square Footage
- Number of Bedrooms
- Number of Bathrooms

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## Machine Learning Model
- Linear Regression

---

## Dataset
The dataset contains housing information such as:
- Living Area
- Bedrooms
- Bathrooms
- Sale Price

Dataset files used:
- train.csv
- test.csv

---

## Project Workflow
1. Data Loading
2. Data Preprocessing
3. Feature Selection
4. Train-Test Split
5. Model Training
6. Prediction
7. Model Evaluation
8. Data Visualization

---

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

---

## Results
The model successfully predicted house prices with a good regression performance.

Example Performance:
- R² Score: 0.63
- MAE: 35788
- MSE: 2806426667

---

## Output Visualization
The project includes a scatter plot comparing:
- Actual Prices
- Predicted Prices
